# Treino-fala
Portal interativo para treino de articulação fonética com reconhecimento de voz em tempo real. Inclui missões lúdicas com feedback visual e auditivo para apoio em terapia da fala, focando-se inicialmente no fonema /l/. Desenvolvido com Web Speech API para uma experiência imersiva e pedagógica no navegador. 🦁🚀
