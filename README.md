Portal de Treino Fonológico 🦁🚀🐍

Este projeto é uma plataforma web interativa desenvolvida para auxiliar no treino da articulação de fonemas (sons da fala). Utiliza a API de Reconhecimento de Voz do navegador para dar feedback em tempo real aos utilizadores.

🌟 Funcionalidades

Reconhecimento de Voz: Validação automática da pronúncia correta através de inteligência do navegador.

Feedback Visual: Emojis animados e barras de progresso que tornam o treino lúdico e motivador para crianças.

Síntese de Voz: Botão de apoio para ouvir a pronúncia correta da palavra (modelo auditivo).

Progressivo: Sistema de "estrelas" para acompanhar o sucesso em cada sessão, com um objetivo de 40 palavras únicas sem repetições.

🎮 Missões Disponíveis

[x] Som L: Treino completo com banco de palavras abrangente (início, meio e fim da palavra).

[ ] Som R: (Planeado: Treino para o R vibrante e brando).

[ ] Som S: (Planeado: Treino para sibilantes).

🛠️ Tecnologias Utilizadas

HTML5 & CSS3: Interface moderna, limpa e totalmente responsiva (Mobile-friendly).

JavaScript (Vanilla): Lógica de jogo, gestão de estado e manipulação do DOM.

Web Speech API: Implementação de SpeechRecognition para captura e SpeechSynthesis para auxílio auditivo.
